I have made a base site using hugo static generator. The posts are written in emacs with org-mode. The plugin `ox-hugo` converts the org file to markdown and places it to the posts folder of the hugo site folder. I have also used roam capture template so that the placeholder for tags and categories are predefined in new org files. The placeholder includes roam_tags, hugo_tags, hugo_categories, HUGO_BASE_DIR path
